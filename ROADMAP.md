# Roadmap

This roadmap is organized as two tracks:

1. Software Engineering Core
2. AI Engineering Specialization

The Software Engineering Core comes first. AI Engineering begins later, after the foundation is strong enough for production-grade AI systems work.

## Track 1: Software Engineering Core

| Order | Topic | Main Evidence |
| ---- | ----- | ------------- |
| 01 | Programming Fundamentals | Notes, small snippets, basic exercises |
| 02 | Computer Science Foundations | Notes, snippets, written explanations |
| 03 | Data Structures and Algorithms | Implementations, tests, problem notes |
| 04 | Git, GitHub, and CLI | Branch workflow, commits, terminal notes |
| 05 | Web Fundamentals | HTML, CSS, JavaScript, HTTP notes |
| 06 | Frontend Engineering | Component-based UI exercises |
| 07 | Backend Engineering | Small API services |
| 08 | Databases | Schema design, queries, migrations |
| 09 | API Design | REST contracts, validation, docs |
| 10 | Auth and Security Basics | Auth flows, secrets handling, threat notes |
| 11 | Testing and Debugging | Unit tests, integration tests, debugging logs |
| 12 | DevOps and Deployment | Docker, deployment notes, health checks |
| 13 | System Design | Architecture notes and ADRs |
| 14 | SWE Portfolio | Polished portfolio projects |

## Track 2: AI Engineering Specialization

| Order | Topic | Main Evidence |
| ---- | ----- | ------------- |
| 01 | Python for AI | Python AI utility scripts |
| 02 | Data Tools | Data cleaning and analysis notebooks or scripts |
| 03 | Math and Statistics | Notes, notebooks, worked examples |
| 04 | Machine Learning | Sklearn project with evaluation |
| 05 | Deep Learning | PyTorch training loop project |
| 06 | Computer Vision | Vision model project |
| 07 | NLP and Transformers | NLP baseline and transformer notes |
| 08 | LLMs and Prompting | Prompt experiments and eval-backed prompts |
| 09 | RAG and Vector Databases | Cited retrieval system |
| 10 | Agents and Tool Use | Bounded tool-using agent |
| 11 | Fine-tuning and Evaluation | Fine-tuning run and eval suite |
| 12 | AI Backend Systems | AI service with streaming and observability |
| 13 | AI Portfolio | Polished AI engineering projects |

## Completion Standard

A topic is only done when its `mastery.md` contains concrete evidence. Reading alone is not enough.
