# AGENTS.md

## Purpose

This roadmap covers Software Engineering Foundations. It is the active roadmap before AI Engineering.

## Rules

- Work in this roadmap first unless the user explicitly asks for another roadmap.
- Notes go in `notes/`.
- Small exercises, drills, and isolated runnable experiments go in `practice/`.
- Larger portfolio work goes in `projects/`.
- References, reading lists, and cheatsheets go in `resources/`.
- Do not mark a topic complete without evidence in `mastery.md`.
- Do not jump to AI topics inside this folder.
- Keep work topic-scoped.
- Preserve existing work and archive unclear material instead of deleting it.
- Do not create frontend, backend, deployment, or web apps unless explicitly asked.
